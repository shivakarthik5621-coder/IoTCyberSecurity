# Unsupervised Network Anomaly Detection using Temporal Convolutional Autoencoder

## Overview

This project presents an **unsupervised anomaly detection framework** based on a **Temporal Convolutional Network (TCN) Autoencoder** for identifying anomalous network traffic.

The model is trained exclusively on benign network traffic and learns normal behavioral patterns. During inference, anomalous traffic is detected using reconstruction errors generated by the autoencoder.

The system eliminates the requirement for labeled attack data during training and is suitable for real-world intrusion detection environments where attack samples are scarce or unavailable.

---

## Key Features

* Unsupervised anomaly detection
* Temporal Convolutional Network (TCN) architecture
* Autoencoder-based feature learning
* Reconstruction error–based anomaly scoring
* Adaptive threshold estimation using percentile statistics
* High scalability for large network datasets
* Visualization of training convergence and detection performance

---

## Model Architecture

### Encoder

The encoder compresses network traffic sequences into a lower-dimensional latent representation.

Input Sequence
↓
Conv1D
↓
TCN Block (dilation = 1)
↓
TCN Block (dilation = 2)
↓
TCN Block (dilation = 4)
↓
Latent Representation

### Decoder

The decoder reconstructs the original sequence from the latent space.

Latent Representation
↓
Conv1D
↓
TCN Block (dilation = 1)
↓
TCN Block (dilation = 2)
↓
TCN Block (dilation = 4)
↓
TimeDistributed Dense Layer
↓
Reconstructed Sequence

### Anomaly Detection

1. Train only on benign traffic.
2. Compute reconstruction errors on validation data.
3. Estimate anomaly threshold using the 99.5th percentile.
4. Flag samples exceeding the threshold as anomalous.

## Dataset

The framework uses two datasets:

### Training Dataset

* Benign network traffic
* Used for learning normal behavior

### Testing Dataset

* Attack traffic
* Used only for evaluation

Preprocessing steps:

* Missing value removal
* Infinite value handling
* Numerical feature selection
* Feature normalization using MinMaxScaler
* Sequence generation using sliding windows
